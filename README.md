# 🚀 Smart PDF Question Extractor - Frontend

## 📌 Overview
This is the frontend of the Smart PDF Question Extractor application.  
It allows users to upload PDF files and view extracted questions in a clean UI.

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- JavaScript

## ✨ Features
- Upload PDF files
- Display extracted questions
- Responsive UI
- Loading animation during processing

## ⚙️ Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/Jitendra-Kumar7078/pdf-extractor-frontend.git
Navigate to project:
cd pdf-frontend
Install dependencies:
npm install
Run the app:
npm start
🔗 Backend API

Make sure backend is running on:

http://localhost:8080
